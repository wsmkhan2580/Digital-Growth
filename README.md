# Digital Growth Agency

A modern portfolio website for a restaurant-focused digital agency built using pure HTML, CSS, and JavaScript.

## ✨ Features

- Fully Responsive Design
- Smooth Scroll Animations
- Live Project Previews
- Services & Pricing Sections
- WhatsApp Contact CTA
- SEO Friendly
- Single File Setup

---

## 📁 Project Structure

```bash
digital-growth-agency/
│

```

## ⚡ Quick Setup

1. Open `index.html`
2. Update your:
   - WhatsApp number
   - Email
   - Pricing
   - Projects
3. Deploy instantly

---

## 🌐 Deployment

### Vercel

1. Upload the project to GitHub
2. Go to vercel.com
3. Import your repository
4. Click Deploy
